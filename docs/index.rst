ReScience C
===============================

The `ReScience C <http://resciencec.theoj.org/>`_ is an online journal that publishes replication and reproducibility projects in computational science.

About this site
---------------

This site contains documentation for authors interested in submitting to ReScience C, reviewers who have generously volunteered their time to review submissions, and editors who manage the ReScience C editorial process.

If you're interested in learning more about ReScience C, you might want to read:

- The `about page <http://resciencec.theoj.org/about>`_ on the main ReScience C site
- `The paper in PeerJ CS <https://doi.org/10.7717/peerj-cs.142>`_ describing the first two years of ReScience C
